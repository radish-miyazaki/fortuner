## fortuner

```
Rust fortune

Usage: fortuner [OPTIONS] <FILE>...

Arguments:
  <FILE>...  Input files or directories

Options:
  -m, --pattern <PATTERN>  Pattern
  -s, --seed <SEED>        Random seed
  -i, --insensitive        Case-insensitive pattern matching
  -h, --help               Print help
  -V, --version            Print version
```
